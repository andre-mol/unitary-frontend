# Unitary — SaaS de Gestão Patrimonial (Front-end)

Aplicação web desenvolvida com foco em **visualização de dados financeiros**, **organização patrimonial** e **experiência de usuário em dashboards SaaS**.

Este projeto simula um sistema completo de gestão de patrimônio, com múltiplas telas, navegação estruturada e interface moderna — utilizando dados mockados para representar um ambiente real.

---

## Demo

https://unitary.com.br

Acesse a versão de demonstração com dados fictícios diretamente pelo site.

---

## Sobre o Projeto

O Unitary foi desenvolvido com o objetivo de:

- Consolidar informações financeiras em uma interface clara
- Facilitar a visualização do patrimônio ao longo do tempo
- Simular um sistema SaaS real com múltiplos módulos
- Explorar boas práticas de UI/UX em dashboards complexos

---

## Tecnologias Utilizadas

- **Next.js**
- **React**
- **Tailwind CSS**
- **JavaScript / TypeScript**
- **Recharts**

---

## Funcionalidades

- Dashboard financeiro com KPIs e Gráficos avançados
- Gráficos de evolução patrimonial
- Gestão de portfólios
- Sistema de metas financeiras
- Controle de orçamento doméstico
- Sistema de notas para ativos
- Linha do tempo financeira
- Tela de autenticação (mockada)

---

## Estrutura do Projeto

O projeto foi estruturado com foco em:

- Componentização reutilizável
- Separação clara entre layout e lógica
- Escalabilidade para múltiplas páginas
- Organização voltada para aplicações SaaS

---

## Decisões Técnicas

- Uso de **Next.js** para estruturação de rotas e performance
- **Tailwind CSS** para rapidez no desenvolvimento e consistência visual
- Dados mockados para simular ambiente real sem dependência de backend
- Layout baseado em dashboards modernos, utilizando de base fintechs

---

## Limitações

- Não possui backend real no site acessível. O desenvolvimento do backend foi pausado devido aos custos de deploy/manutenção
- Não há persistência de dados
- Autenticação simulada

---

## Melhorias Futuras

- Integração com backend
- Persistência de dados do usuário
- Integração com APIs financeiras
- Otimizações de performance e caching
